URL Slug Optimizer

Enhance your website's SEO and user experience with the URL Slug Optimizer.

Key Features

  • ✅ Custom Slug Creation: Replace default Odoo slugs with SEO-friendly URLs.
  • ✅ Original Route Preservation: Maintain existing links and website functionality.
  • ✅ Redirection Management: Prevent 404 errors with automatic redirects.
  • ✅ Easy-to-Use Interface: Manage slugs and redirects within Odoo.
