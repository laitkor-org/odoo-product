URL Slug Optimizer

Enhance your website's SEO and user experience with the URL Slug Optimizer.

Key Features

  • ✅ Custom Slug Creation: Replace default Odoo slugs with SEO-friendly URLs.
  • ✅ Original Route Preservation: Maintain existing links and website functionality.
  • ✅ Redirection Management: Prevent 404 errors with automatic redirects.
  • ✅ Easy-to-Use Interface: Manage slugs and redirects within Odoo.

User Guide

Getting Started

1. Install the module in your Odoo system.
2. Access the dashboard and click "New" to create a new URL mapping.

Creating a URL Mapping

Enter the original and updated URLs. Choose whether to fetch content or redirect.

✅ Correct Format:
/shop/bougainvillea-glabra-bougainvillea-73#attr=45
❌ Incorrect Format:
https://stage.rootsraja.in/shop/bougainvillea-glabra-bougainvillea-73#attr=45
Batch Delete Functionality

Select multiple URLs, click settings, and choose Delete.

Install the Module - Begin by installing the URL Slug Optimizer module in your Odoo system.

Access the Dashboard - Once installed, you will land on the dashboard page. Click on the "New" button to create a new URL mapping.

–

After clicking "New", you will be redirected to a page where you can:

  • Enter the Original URL
  • Enter the Updated URL
  • Choose whether to fetch content from the updated URL or simply redirect (default option).

How Fetch Content Works

If "Fetch Content" is Unchecked: The system will redirect to the updated URL without fetching content.

▶️ Example Video

If "Fetch Content" is Checked: The system will dynamically fetch and display content from the updated URL.

Batch Delete Functionality

On the dashboard, you can also delete multiple URL mappings in bulk:

  • Select the URLs you want to delete.
  • Click on the Settings icon.
  • Choose Delete to remove the selected URLs.

This feature helps manage and optimize URLs efficiently.
